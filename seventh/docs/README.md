## How it all fits together
1. **Prep** — Teaching Plan *Part 0* (run the demo, payment sandboxes, deploy for marketing).
2. **Hand out** the four student docs.
3. **Teach** the 6 sessions from the Teaching Plan (Build → Marketing).
4. **Students build** their own store, using the LearningMart demo + the "demo file → question" map as reference.
5. **Assess** — printed report answering the 16 questions + viva (Lab Work & Viva).

## The demo
[`../demo`](../demo) — run it on XAMPP/MAMP (see Teaching Plan Part 0 or `../demo/README.md`). It covers
every lab question end-to-end and is the students' reference implementation.
